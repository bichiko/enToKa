jQuery plugin for Georgian Keyboard

Supported browsers: Google Chrome, Mozilla Firefox, Internet Explorer, Opera, Microsoft Edge

Do you want to implement ENGtoGEO in your website?

You need to include link in your document

and paste inside body $('input').geoKbd('.switcher')

$('.input') is a Jquery selector and by binding geoKbd method it turns into georgian keyboard. And parameter for geoKbd is switcher checkbox className

In this example $('input').geoKbd('.switcher') we have to have at lease one input element and checkbox with classname .switcher

NB! don;t forget to include Jquery CDN link as well( before entoka API)
